# TODO-APP

A clean and easy-to-use Todo application developed using **HTML**, **CSS**, and **JavaScript**. Users can add tasks, delete tasks, and manage their day-to-day activities efficiently with a simple UI.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Lightweight and responsive UI
- Simple and beginner-friendly code structure

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Open index.html in your browser.
